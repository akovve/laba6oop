Detailed Description

template<typename Key, typename Value>
class MyDictionary< Key, Value >

Шаблонный класс словаря с уникальными ключами.

Реализован вручную на динамическом массиве пар ключ-значение. При добавлении пары с уже существующим ключом добавление не происходит.

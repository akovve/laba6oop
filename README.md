## Задание 1: Множество уникальных элементов (MySet)

**Условие:**
Создайте класс, реализующий функционал неупорядоченного списка (обратиться к элементу по индексу нельзя), который может содержать только уникальные элементы (в случае добавления повторяющихся значений ошибки нет, такой элемент просто не добавляется). Полем класса является массив. В классе должны присутствовать методы добавления и удаления элементов, а также методы:
- `Union()` — объединяет две коллекции
- `Except()` — позволяет удалить из первой коллекции элементы второй
- `Intersect()` — получает коллекцию, состоящую из элементов, общих для двух коллекций
- `Contains()` — проверяет на наличие элемента, возвращает bool

**Задача варианта:**
Есть перечень дискотек города. Студенты группы любят посещать дискотеки. Известно для каждого студента, в каких дискотеках он побывал. Определить:
- в какие дискотеки из перечня ходили все студенты группы
- в какие дискотеки из перечня ходили некоторые студенты группы
- в какие дискотеки из перечня не ходил ни один из студентов группы

### Класс MySet:

**Поля (private):**
- `data_` — `std::vector<T>` для хранения уникальных элементов

**Конструкторы:**
- `MySet()` — создает пустое множество

**Методы (public):**
- `Add(const T& item)` — добавляет элемент (только если его нет)
- `Remove(const T& item)` — удаляет элемент
- `Contains(const T& item) const` — проверяет наличие элемента
- `Union(const MySet& other) const` — объединение множеств
- `Except(const MySet& other) const` — разность множеств
- `Intersect(const MySet& other) const` — пересечение множеств
- `Size() const` — возвращает размер множества
- `begin()`, `end()` — итераторы для обхода

### Алгоритм работы кода:

1) Создается перечень всех дискотек города: "Рай", "Метелица", "Пилот", "Космос", "Юность"
2) Создаются множества посещений для трех студентов
3) Вычисляются:
   - Пересечение всех множеств (дискотеки, где были все)
   - Объединение всех множеств (дискотеки, где были некоторые)
   - Разность общего перечня и объединения (дискотеки, где не был никто)
4) Результаты выводятся на экран

## Задание 2: Словарь (MyDictionary)

**Условие:**
Опишите класс, который реализует функционал словаря (запрещается использовать готовые решения). В классе должны присутствовать методы добавления и удаления элементов, учесть, что ключи должны быть уникальными. Исходные данные содержатся в текстовом файле.

**Задача варианта:**
В некотором вузе абитуриенты проходили предварительное тестирование. Тестирование проводится по трём предметам, по каждому предмету абитуриент может набрать от 0 до 100 баллов. К сдаче экзаменов в первом потоке допускаются абитуриенты, набравшие не менее 30 баллов по каждому предмету, причём сумма баллов должна быть не менее 140. На вход подаются сведения о результатах тестирования. Вывести фамилии и имена абитуриентов, допущенных к сдаче экзаменов, в алфавитном порядке.

### Класс MyDictionary:

**Поля (private):**
- `data_` — динамический массив пар ключ-значение
- `size_` — текущее количество элементов
- `capacity_` — вместимость массива

**Конструкторы:**
- `MyDictionary()` — создает пустой словарь
- `MyDictionary(const MyDictionary& other)` — конструктор копирования
- `~MyDictionary()` — деструктор

**Методы (public):**
- `operator=` — оператор присваивания
- `Add(const Key& key, const Value& value)` — добавляет пару ключ-значение
- `Remove(const Key& key)` — удаляет элемент по ключу
- `ContainsKey(const Key& key) const` — проверяет наличие ключа
- `TryGetValue(const Key& key, Value& value) const` — получает значение по ключу
- `Size() const` — возвращает размер словаря

**Методы (private):**
- `Reserve(int new_capacity)` — изменяет вместимость массива
- `FindIndex(const Key& key) const` — находит индекс элемента по ключу

### Структура Applicant:

**Поля:**
- `surname` — фамилия абитуриента (строка)
- `name` — имя абитуриента (строка)
- `scores[3]` — массив баллов по трём предметам

**Операторы:**
- `operator<` — для сортировки по фамилии, затем по имени

### Алгоритм работы кода:

1) Открывается файл `applicants.txt` с данными абитуриентов
2) Читается количество абитуриентов N
3) Для каждого абитуриента читается строка формата: `<Фамилия> <Имя> <балл1> <балл2> <балл3>`
4) Проверяются условия допуска:
   - каждый балл ≥ 30
   - сумма баллов ≥ 140
5) Подходящие абитуриенты добавляются в вектор
6) Вектор сортируется по фамилии и имени
7) Результат выводится на экран

### Формат входного файла (applicants.txt):

```
3
Иванов LL 45 67 30
Петров WWW 29 100 50
Сидоров Tewiq 40 40 70
```

## Задание 3: Трехмерная точка

**Условие:**
Создайте подвид сущности Точка, которая будет иметь не две, а три координаты на плоскости: X, Y, Z, и возможность подсчета расстояния между двумя трехмерными точками.

### Класс Point2D:

**Поля (private):**
- `x_` — координата X
- `y_` — координата Y

**Конструкторы:**
- `Point2D(double x = 0.0, double y = 0.0)` — создает точку с заданными координатами

**Методы (public):**
- `GetX()`, `GetY()` — возвращают координаты
- `SetX()`, `SetY()` — устанавливают координаты
- `DistanceTo(const Point2D& other) const` — вычисляет расстояние до другой точки

### Класс Point3D (наследник Point2D):

**Поля (private):**
- `z_` — координата Z

**Конструкторы:**
- `Point3D(double x = 0.0, double y = 0.0, double z = 0.0)` — создает точку с заданными координатами

**Методы (public):**
- `GetZ()` — возвращает координату Z
- `SetZ()` — устанавливает координату Z
- `DistanceTo(const Point3D& other) const` — вычисляет расстояние до другой трехмерной точки (переопределение)

### Алгоритм работы кода:

1) Создаются две трехмерные точки: p1(1,2,3) и p2(4,6,8)
2) Вычисляется расстояние между ними
3) Создаются точки p3(0,0,0) и p4(3,4,12)
4) Вычисляется расстояние между ними
5) Результаты выводятся на экран
